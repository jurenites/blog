# Drupal Content Model

## Timeline Event

Purpose: turn the CV into an interactive timeline with bookmarks.

Fields:

- Title
- Slug
- Date start
- Date end
- Company / client
- Public URL
- Archived URL
- Role
- Project type
- Technology tags
- Drupal/CMS/framework version
- NDA status: public, anonymized, private
- Company logo
- Project icon
- Hero image
- Short summary
- What happened at this moment
- My contribution
- What I learned
- Related artifacts
- Weight / importance
- Featured bookmark

## Project

Purpose: portfolio pages for projects that can be public or anonymized.

Fields:

- Title
- Date range
- Client/company
- Role
- Public status
- Summary
- Problem
- Process
- Result
- Screenshots/artifacts
- Related timeline events
- Related articles

## Article

Purpose: blog posts and long-form analysis.

Fields:

- Title
- Slug
- Teaser
- Body
- Topic
- Related projects
- Related external references
- Hero image
- Publish state

## Gallery Item

Purpose: visual inspiration, GIF/image/code recreation entries.

Fields:

- Title
- Source URL
- Original author
- Source status
- Media
- Code recreation link
- Notes
- Tags

## External Reference

Purpose: designers, videos, tweets/posts, tools, and links that shape the project.

Fields:

- Title
- URL
- Author/account
- Platform
- Topic
- Notes
- Related article/project

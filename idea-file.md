# Idea File

This is the rough working area for the project. Stable decisions should move into `docs/` or `src/token/tokens.yaml`.

## Product

Create a personal Drupal 11 website for jurenites. It should be a blog, portfolio, interactive CV, and public process archive.

The site should be direct and personal. It should show how I think about product, UI, CMS structure, naming, data models, and implementation.

## Pages

- About
- Work Timeline
- Portfolio
- Writing
- Culture
- UI Reviews
- Gallery
- Contact / Contact Me

## Signature Features

- Pixel-dithered orange gradient background.
- Interactive CV timeline with slider bookmarks.
- Company/project logos on timeline events.
- Storybook connected to actual Drupal theme components.
- Public design/process cookbook.
- Progressive image/GIF/media loading component.
- Personal contact widget with avatar.

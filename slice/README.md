# Slice Theme Source

This folder contains editable frontend source for the Drupal theme.

- SCSS source: `slice/src/scss/`
- JavaScript source: `slice/src/js/`
- Drupal theme output: `web/themes/custom/jurenites_theme/css/style.min.css` and `web/themes/custom/jurenites_theme/js/script.min.js`

Run:

```sh
npm run build:theme
```

The Drupal theme should keep Twig templates and generated minified assets. Edit source here, then rebuild.
